# 🎬 ShortFlixApp - Short-Form Video Streaming Mobile Web Platform

An innovative, high-performance **Short-Form Video Streaming Application** built using **.NET 10** and **Blazor Web (Interactive Server)**. Designed specifically for vertical, micro-drama content (1-3 minute episodes) inspired by industry leaders like ReelShort and DramaBox.

---

## 🪙 Monetization & Business Model
This project is engineered with a fully functional commercial ecosystem to demonstrate real-world product management standards:
1. **Pay-per-Episode (In-App Coins):** Users watch the first episodes for free, then unlock subsequent episodes utilizing a virtual digital currency wallet ($2.99 for 200 coins).
2. **VIP Subscription Pass:** A premium monthly tier ($7.99/mo) that grants immediate, ad-free access to the entire streaming catalog.

---

## 🚀 Key Technical Features
* **Modern Cinematic UI:** Implements a premium Dark-Theme layout optimized for mobile screens and fluid responsive scaling.
* **State Management:** Live, dynamic updates of the user's digital wallet and account tier status via C# reactive logic.
* **Component-Driven Architecture:** Highly scalable and reusable Blazor component structures.

---

## 🛠️ Tech Stack
* **Framework:** .NET 10.0 (Blazor Web App)
* **Language:** C#
* **Frontend:** HTML5, CSS3 (Mobile-First Layout)
